# CMS Publisher

Publiceer blogartikelen en pagina's direct naar WordPress of Webflow via een chatbericht. Je assistent formatteert de content, voegt metadata toe en zet alles klaar voor publicatie.

---

## Wat kan je assistent?

- Blogartikelen publiceren naar WordPress (als concept of direct live)
- Pagina's aanmaken in Webflow
- SEO-titel, meta-omschrijving en slug automatisch invullen
- Content als concept opslaan voor review
- Bestaande berichten bijwerken

---

## Vereisten

- **Plan:** Growth of hoger
- **Integraties:** WordPress (Application Password) of Webflow (API Token + Collection ID)

---

## Hoe activeer je CMS Publisher?

1. Ga naar **Dashboard → Skills**
2. Klik op **"Activeer"** bij CMS Publisher
3. Ga naar **Dashboard → Instellingen → Integraties**
4. Kies je platform: WordPress of Webflow
5. Voer je credentials in (zie hieronder)
6. De skill is direct actief

**WordPress:** Maak een Application Password aan via *Gebruikers → Profiel → Application Passwords* in je WordPress admin.

**Webflow:** Genereer een API Token in je Webflow-projectinstellingen. Voeg ook je Collection ID in.
