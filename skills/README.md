# Skills marketplace

Skills zijn uitbreidingen voor je AI-werknemer. Met een skill geef je hem extra mogelijkheden bovenop de basisfuncties.

---

## Actieve skills

### Eigen Telegram Bot

**Slug:** `own-bot` | **Categorie:** Integratie | **Prijs:** €2,99/mnd

Standaard maakt je assistent gebruik van de gedeelde dGENIX bot. Met de **Eigen Bot** skill gebruik je een eigen Telegram bot — met je eigen naam en profielfoto.

**Wat heb je nodig?**
1. Een Telegram-account
2. Een bot aangemaakt via [BotFather](https://t.me/BotFather) (`/newbot`)
3. Je bot token (geeft BotFather aan je na aanmaak)

**Hoe activeer je het?**
1. Ga naar **Dashboard → Skills**
2. Klik op **"Activeer"** bij Eigen Telegram Bot
3. Vul je bot token en Telegram chat ID in
4. Klik op **"Opslaan"** — je assistent schakelt automatisch over naar je eigen bot

**Voordelen:**
- Je bot heeft je eigen naam (bijv. "Mijn AI-assistent")
- Klanten/contacten zien jouw branding, niet dGENIX
- Volledig je eigen kanaal

## Hoe werken skills?

1. Ga naar **Dashboard → Skills**
2. Klik **"Activeer"** bij een skill
3. Je betaalt een maandelijks bedrag bovenop je abonnement (via Stripe)
4. De skill wordt direct actief in je assistent

Je kunt een skill op elk moment deactiveren via **Dashboard → Skills → "Verwijder"**. De skill blijft actief tot het einde van de lopende betaalperiode.
