# Dagplanner & Life Assistant

De Dagplanner geeft je AI-werknemer inzicht in je dag — op basis van je agenda, taken en persoonlijke doelen.

---

## Wat kan je assistent?

- Een dagoverzicht opstellen (afspraken + taken gecombineerd)
- Taken prioriteren op urgentie en impact
- Je dagindeling voorstellen op basis van je werkpatroon
- Reminders sturen voor belangrijke momenten

Als Google Calendar gekoppeld is, worden je afspraken automatisch meegenomen in het overzicht.

---

## Voorbeeldopdrachten

```
Wat staat er vandaag op mijn agenda?
```
```
Prioriteer mijn taken voor deze week
```
```
Plan mijn ochtend: ik wil twee uur focuswerk inbouwen voor mijn meeting om 14:00
```
```
Stel een dagschema voor op basis van mijn lopende taken
```

---

## Hoe activeer je Dagplanner?

1. Ga naar **Dashboard → Skills**
2. Klik op **"Activeer"** bij Dagplanner & Life Assistant
3. Direct actief — geen extra koppeling vereist

**Optioneel:** Koppel Google Calendar voor een completer dagoverzicht.
→ [Google Calendar koppelen](../integraties/google-calendar.md)
