# Stripe Inzichten

Vraag omzetdata, betalingsinformatie en klantgegevens op uit je Stripe-account — direct via een chatbericht. Geen dashboard openen, geen rapporten exporteren.

---

## Wat kan je assistent?

- Dagelijks of wekelijks omzetoverzicht opvragen
- Mislukte betalingen identificeren en opvolgen
- Klantinformatie ophalen op basis van e-mail of ID
- MRR, churn en actieve abonnementen monitoren
- Specifieke transacties raadplegen

---

## Vereisten

- **Plan:** Growth of hoger
- **Integraties:** Stripe Restricted API Key (alleen lees-rechten)

---

## Hoe activeer je Stripe Inzichten?

1. Ga naar **Dashboard → Skills**
2. Klik op **"Activeer"** bij Stripe Inzichten
3. Ga naar **Dashboard → Instellingen → Integraties**
4. Maak in je Stripe Dashboard een **Restricted Key** aan met lees-rechten
5. Voer de API-sleutel in bij dGENIX
6. De skill is direct actief

---
