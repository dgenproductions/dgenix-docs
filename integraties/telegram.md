# Telegram verbinden

Telegram is een extra interface naast het dGENIX dashboard — beide kanalen zijn synchroon gekoppeld.

---

## Verbinden

1. Installeer [Telegram](https://telegram.org) op je telefoon of computer
2. Ga naar **Settings → Telegram**
3. Klik **Verbinden** en volg de stappen (Login Widget of chat ID)
4. Stuur `/start` naar [@dGENIXbot](https://t.me/dGENIXbot) om je assistent te activeren

---

## Commando's

| Commando | Functie |
|----------|---------|
| `/start` | Assistent activeren en welkomstbericht ontvangen |
| `/credits` | Huidig credit saldo opvragen |
| `/help` | Snelle lijst van beschikbare commando's |

---

## Spraakberichten

Stuur een spraakbericht via Telegram — de assistent transcribeert het automatisch en verwerkt het als tekst. Je hoeft niets extra's te doen.

---

## Web ↔ Telegram sync

Berichten die je via de **AI Assistent pagina** in de web-app stuurt, worden doorgestuurd naar Telegram (en omgekeerd). Je gesprekgeschiedenis is altijd gesynchroniseerd tussen web en Telegram.

---

## Eigen Telegram Bot (optioneel)

Via de **Eigen Bot** add-on in de Skills Marketplace gebruik je een eigen bot token. Voordelen:
- Eigen naam en profielfoto voor je assistent in Telegram
- Volledig gepersonaliseerde chatervaring

Activeren:
1. Maak een bot aan via [@BotFather](https://t.me/BotFather) in Telegram
2. Kopieer het bot token
3. Activeer de Eigen Bot add-on via Skills Marketplace
4. Voer het token in via Settings → Telegram → Eigen bot

---

## Problemen oplossen

**Assistent reageert niet:**
- Controleer of Telegram correct gekoppeld is via Settings → Telegram
- Gebruik de "Assistent herstarten" knop
- Controleer je credit saldo (Credits-chip rechtsboven in de app)

**Koppeling kwijt:**
- Ga naar Settings → Telegram en verbind opnieuw

**Andere problemen:**
- Open een support ticket via Help & Support in het dashboard
