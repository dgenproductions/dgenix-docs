# Connectors

Connectors zijn externe koppelingen die je aan je AI-assistent toevoegt. Via **Dashboard → Connectors** koppel je je tools eenmalig — daarna gebruikt je assistent ze automatisch bij relevante opdrachten.

---

## Hoe het werkt

1. Ga naar **Dashboard → Connectors**
2. Kies de connector die je wil koppelen
3. Doorloop de autorisatiestap (OAuth of API-sleutel invoeren)
4. De connector is direct beschikbaar voor alle bijbehorende skills

## OAuth vs. API-sleutel

**OAuth (klik + autoriseer):** Gmail, Google Calendar, Google Drive, Google Sheets, Slack, Notion, LinkedIn, Microsoft 365, Canva

**API-sleutel (token invoeren):** WhatsApp Business, Instagram Business, Stripe, HubSpot
